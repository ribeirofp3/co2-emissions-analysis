# CO2-Emissions-Analysis
Data analysis project exploring global CO2 emission trends and per capita intensity using Python (Pandas and NumPy).

## Project Overview

At this project the dataset GCB (Global Carbon Project) could provide us sobre depth analysis o fossil CO2 emissions. The real objective was to track historical emission trends, like identifying key inflection points sush as the post-pandemic recovery of 2021, and compare emission intensities between nations.

## Technologies Used
- Python (Pandas, NumPy, Matplotlib)
- Jupyter Notebook / Google Colab

## Technical Skills Demonstrated
- **Data Wrangling:** Handling missing values, cleaned large datasets and filtering for relevant entities.
- **Data Manipulation:** Used `Pandas` for aggregation and `NumPy` for time-series differentiation (`np.diff`) to calculate growth rates.
- **Visualization:** Created temporal trend charts and comparative analysis plots using basic `Matplotlib`.

## Key Insights
- **The Great Acceleration:** Having a correlation with industrial expansion, there is a identification of a significant surge in global emissions starting from 1950. 
- **Turning Point 2021:** Analysis of the post-pandemic economic rebound,. Using NumPy derivates was possibible to highlighting the record-breaking in CO2 emissions.
- **Consumption Patterns:** A comparative analysis could show us that while some nations lead in total emissions (China), tohe can show significantly higher per capita emissions (lifestyle/efficiency). emphasizing the need for nuanced climate policy.
