# Co2-Emissions-Analysis
Data analysis project exploring global CO2 emission trends and per capita intensity using Python (Pandas and Numpy).

## Project Overview
  
At this project the dataset GCB (Global Carbon Project) could provide us sobre depth analysis o fossil CO2 emissions. The real objective was to track historical emission trends, like identifying key inflection points sush as the post-pandemic recovery of 2021, and compare emission intensities between nations.
